# User Stories

- As a user, I want to log in so I can view my accounts.
- As a user, I want to see my balance so I know how much money I have.
- As a user, I want to deposit money so my balance increases.
- As a user, I want to withdraw money so my balance decreases.
- As a user, I want to transfer money between accounts.
- As a user, I want to see my transaction history.
- As a user, I want error messages when I enter invalid information.
